pg-listener

Reliable notification listener for PostgreSQL.

## Installing

```
$ npm install pg-listener
```

## Testing

First, clone the repository and install DEV dependencies.

```
$ npm test
```

Testing with coverage:
```
$ npm run coverage
```

## Usage

```js
var listen = require('pg-listener');
```
